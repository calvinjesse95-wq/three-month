
HOW TO USE THIS ANNIVERSARY SURPRISE

1. Put your photos in this folder
   - Rename them to:
     photo1.jpg
     photo2.jpg

2. Download the song:
   Ed Sheeran - Perfect
   Rename it to:
   perfect.mp3
   Put it in this same folder.

3. Open index.html in a browser
   OR upload the folder to Netlify / GitHub Pages and send the link on WhatsApp.

4. To add more photos:
   Copy this line and paste it again inside index.html:
   <div class="bubble"><img src="photo3.jpg" class="photo"></div>
